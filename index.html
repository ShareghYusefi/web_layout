Web Layout

Traditional flow of HTML elements are to be staclked on top of each other. These types of elements are know as block elements and they start on a new line.

Inline elements are those that flow with the text and do not start on a new line. A good example is the span tag. These elements do not maintain their size and shape.

Inline-block elements are those that flow with the text and the also maintain their size and shape.
